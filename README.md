# Route Weather App

A web application that checks real-time weather conditions and evaluates transport feasibility between two locations.

## Features
- Start and destination weather information
- Approximate distance calculation
- Transport mode feasibility (Two-Wheeler, Car, Train, Plane)
- Weather-based safety alerts

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
